# Անիպեմզայի դպրոցի կայք

Պաշտոնական տեղեկատվական կայք Անիպեմզայի միջնակարգ դպրոցի համար։  
Կառուցված է **Next.js**-ով, աջակցում է 3 լեզու՝ հայերեն, ռուսերեն, անգլերեն։

---

## Ադմինի վահանակ

### Մուտք

Հասցե՝ `https://[կայքի-հասցե]/admin`

Մուտք գործելու համար անհրաժեշտ է **մուտքանուն** (login) և **գաղտնաբառ** (token),  
որոնք տրամադրվում են ծրագրավորողի կողմից։

---

### Ինչ կարող է ադմինը անել

Ադմինի վահանակից կարող ես **ավելացնել, խմբագրել և հեռացնել նյութեր** (entries)  
կայքի ցանկացած բաժնի ցանկացած ենթաէջի համար։

#### Նյութ ավելացնելիս լրացվում է.

| Դաշտ | Բացատրություն |
|---|---|
| **Բաժին** | Ո՞ր բաժնում կհայտնվի (Ուսուցիչներ, Մատակարարում, Կապ...) |
| **Ենթաէջ** | Կոնկրետ ենթաէջ (Ղեկավարություն, Ծրագրեր, Հայտարարություններ...) |
| **Վերնագիր** | Նյութի անվանումը |
| **Տեքստ** | Նյութի բովանդակությունը (ազատ տեքստ) |
| **Ֆայլեր** | PDF, Word, Excel և այլ ֆայլեր վերբեռնել ուղղակի ադմինից |
| **Հղումներ** | Արտաքին հղումներ ձեռքով ավելացնել (ըստ ցանկության) |

#### Ֆայլերի վերբեռնում

Ֆայլեր (PDF, Word, Excel, նկարներ) կարելի է **ուղղակի վերբեռնել** ադմինի վահանակից։  
Ոչ մի ծրագրի կամ FTP-ի կարիք չկա — ֆայլն ավտոմատ կպահվի կայքում և  
կհայտնվի համապատասխան ենթաէջում։

#### Փնտրտուք

Ցանկում կա **որոնման** հնարավորություն — կարող ես գտնել ավելացրած ցանկացած նյութ  
անվան կամ բաժնի գծով։

---

### Ինչ **չի կարող** փոխել ադմինը

Հետևյալ տվյալները **ծրագրի ֆայլերում** են՝ դրանց փոփոխության համար  
անհրաժեշտ է ծրագրավորողի օգնությունը.

| Ինչ | Որտեղ |
|---|---|
| Դպրոցի անուն, հասցե, հեռախոս, էլ. փոստ | `school.config.ts` |
| Facebook, YouTube հղումներ | `school.config.ts` |
| Քարտեզի կոորդինատներ | `school.config.ts` |
| Տնօրենի անուն, պաշտոն, լուսանկար | `app/.../staff/leadership/page.tsx` + `public/director.jpg` |
| Դպրոցի պատմության տեքստ | `app/.../about/history/page.tsx` |
| Կայքի logo և hero-նկար | `public/logo.png`, `public/school.jpg` |
| Նավիգացիայի բաժիններ ու ենթաէջեր | `app/data.ts` |

## Տեղական գործարկում (ծրագրավորողի համար)

```bash
# 1. Կախվածությունների տեղադրում
npm install

# 2. Env ֆայլի ստեղծում
cp .env.example .env.local
# Լրացնել ADMIN_LOGIN, ADMIN_TOKEN դաշտերը

# 3. Dev server-ի գործարկում
npm run dev
# Կայքը՝ http://localhost:3000/hy
# Ադմին՝   http://localhost:3000/admin
```

### Prod build

```bash
npm run build
npm start
```

## Ֆայլերի կառուցվածք

```
school.config.ts        ← Դպրոցի բոլոր տվյալները (1 ֆայլ)
app/data.ts             ← Նավիգացիայի բաժիններ
data/materials.json     ← Ադմինի կողմից ավելացված նյութեր
public/                 ← Նկարներ (director.jpg, school.jpg, logo.png...)
messages/
  hy.json               ← Հայերեն UI տեքստեր
  ru.json               ← Ռուսերեն UI տեքստեր
  en.json               ← Անգլերեն UI տեքստեր
app/[locale]/section/   ← Կայքի բոլոր ենթաէջերը
app/admin/              ← Ադմինի վահանակ
```

## Լեզուներ

Կայքը հասանելի է 3 URL-ով.

| Լեզու | URL |
|---|---|
| Հայերեն (default) | `/hy/...` |
| Ռուսերեն | `/ru/...` |
| Անգլերեն | `/en/...` |
